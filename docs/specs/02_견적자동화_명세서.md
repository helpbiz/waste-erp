# 💰 견적 자동화 도구 명세서

> **목표**: 신청서의 직원·차량 수를 입력하면 즉시 표준 견적서(PDF)가 자동 생성·발송되는 도구.
> **참조**: 서비스_개설_절차.md 부록 B "요금제 + SLA"
> **상태**: 명세 단계
> **우선순위**: 🔥 High (영업 응답 1일 → 5분 단축)

---

## 1. 목적

| 항목 | 내용 |
|---|---|
| Why | 영업 매니저가 매번 수기 산정 → 자동 견적으로 응답 시간 단축 |
| Who | 운영팀(슈퍼관리자) + 신청업체(메일 수신) |
| What | 신청서 → 자동 요금 산정 → 견적서 PDF 생성 → 자동 발송 |
| Success | 신청 접수 후 **5분 내 견적서 발송** (현재 1~2일) |

## 2. 요금 산정 로직

### 2.1 플랜 자동 추천 (Score 기반)

```
입력값:
  - totalEmployees (전체 직원 수)
  - totalVehicles (전체 차량 수)
  - municipalityCount (관할 지자체 수)

추천 플랜:
  if (totalEmployees ≤ 30 && totalVehicles ≤ 10):
    → STARTER (50만/월)
  
  elif (totalEmployees ≤ 100 && totalVehicles ≤ 30):
    → STANDARD (150만/월)
  
  elif (totalEmployees ≤ 300 && totalVehicles ≤ 100):
    → PRO (350만/월)
  
  else:
    → ENTERPRISE (별도 협상, 기본 600만/월)
```

### 2.2 기본 요금 (월정액)

| 플랜 | 기본료 | 직원 한도 | 차량 한도 | 추가 직원/명 | 추가 차량/대 |
|---|---|---|---|---|---|
| STARTER | 500,000원 | 30명 | 10대 | 10,000원 | 30,000원 |
| STANDARD | 1,500,000원 | 100명 | 30대 | 8,000원 | 25,000원 |
| PRO | 3,500,000원 | 300명 | 100대 | 6,000원 | 20,000원 |
| ENTERPRISE | 협상 | 무제한 | 무제한 | 협상 | 협상 |

### 2.3 부가 서비스 (1회성 또는 옵션)

| 항목 | 비용 | 자동 추가 조건 |
|---|---|---|
| 데이터 마이그레이션 | 1,000,000원 | needsMigration === true |
| 자체 도메인 셋업 | 1,000,000원 | desiredPlan === 'PRO' 이상 + 명시적 요청 |
| 추가 운영자 교육 | 500,000원 | totalEmployees > 100 시 권장 |
| 24/7 전담 지원 (월) | 1,000,000원 | 옵션 |
| 자체 호스팅 | 별도 협상 | desiredPlan === 'ENTERPRISE' |

### 2.4 할인 정책

| 조건 | 할인율 |
|---|---|
| 1년 선납 | -10% |
| 다년 계약 (3년+) | -15% |
| 신규 도입 첫 6개월 | -20% (런칭 프로모션) |
| 다수 지자체(3개+) 통합 운영 | -10% |

## 3. 핵심 함수 구현

```typescript
// lib/quote.ts

export type QuoteInput = {
  totalEmployees: number;
  totalVehicles: number;
  needsMigration?: boolean;
  needsCustomDomain?: boolean;
  needs247Support?: boolean;
  contractYears?: number;
  isLaunchPromo?: boolean;
  multiMunicipality?: boolean;
};

export type QuoteOutput = {
  recommendedPlan: 'STARTER' | 'STANDARD' | 'PRO' | 'ENTERPRISE';
  monthlyBase: number;          // 기본 월정액
  monthlyExtra: number;         // 추가 직원·차량 비용
  monthlyTotal: number;         // 월 총액 (base + extra)
  oneTimeFees: { item: string; cost: number }[];
  oneTimeTotal: number;
  discounts: { reason: string; rate: number; amount: number }[];
  discountedMonthly: number;    // 할인 적용 월 총액
  yearlyTotal: number;          // 12개월 총액 (할인 적용)
  breakdown: string[];          // 사람이 읽기 좋은 설명
};

const PLAN_RATES = {
  STARTER:    { base: 500_000,   empLimit: 30,  vehLimit: 10,  empOver: 10_000, vehOver: 30_000 },
  STANDARD:   { base: 1_500_000, empLimit: 100, vehLimit: 30,  empOver: 8_000,  vehOver: 25_000 },
  PRO:        { base: 3_500_000, empLimit: 300, vehLimit: 100, empOver: 6_000,  vehOver: 20_000 },
  ENTERPRISE: { base: 6_000_000, empLimit: Infinity, vehLimit: Infinity, empOver: 5_000, vehOver: 15_000 },
};

export function calculateQuote(input: QuoteInput): QuoteOutput {
  // 1. 플랜 추천
  let plan: QuoteOutput['recommendedPlan'];
  if (input.totalEmployees <= 30 && input.totalVehicles <= 10) plan = 'STARTER';
  else if (input.totalEmployees <= 100 && input.totalVehicles <= 30) plan = 'STANDARD';
  else if (input.totalEmployees <= 300 && input.totalVehicles <= 100) plan = 'PRO';
  else plan = 'ENTERPRISE';
  
  const rate = PLAN_RATES[plan];
  
  // 2. 기본료 + 초과분
  const empOver = Math.max(0, input.totalEmployees - rate.empLimit);
  const vehOver = Math.max(0, input.totalVehicles - rate.vehLimit);
  const monthlyExtra = empOver * rate.empOver + vehOver * rate.vehOver;
  const monthlyTotal = rate.base + monthlyExtra;
  
  // 3. 1회성 비용
  const oneTimeFees: { item: string; cost: number }[] = [];
  if (input.needsMigration) oneTimeFees.push({ item: '데이터 마이그레이션', cost: 1_000_000 });
  if (input.needsCustomDomain) oneTimeFees.push({ item: '자체 도메인 셋업', cost: 1_000_000 });
  if (input.totalEmployees > 100) oneTimeFees.push({ item: '추가 운영자 교육', cost: 500_000 });
  const oneTimeTotal = oneTimeFees.reduce((s, f) => s + f.cost, 0);
  
  // 4. 247 지원 옵션 (월간 추가)
  const support247 = input.needs247Support ? 1_000_000 : 0;
  const monthlyWithSupport = monthlyTotal + support247;
  
  // 5. 할인 누적
  const discounts: { reason: string; rate: number; amount: number }[] = [];
  let discountSum = 0;
  
  if (input.contractYears && input.contractYears >= 3) {
    discounts.push({ reason: '다년 계약(3년+)', rate: 0.15, amount: monthlyWithSupport * 0.15 });
    discountSum += 0.15;
  } else if (input.contractYears === 1) {
    discounts.push({ reason: '1년 선납', rate: 0.10, amount: monthlyWithSupport * 0.10 });
    discountSum += 0.10;
  }
  
  if (input.isLaunchPromo) {
    discounts.push({ reason: '런칭 프로모션 (첫 6개월)', rate: 0.20, amount: monthlyWithSupport * 0.20 });
    discountSum += 0.20;
  }
  
  if (input.multiMunicipality) {
    discounts.push({ reason: '다수 지자체 통합', rate: 0.10, amount: monthlyWithSupport * 0.10 });
    discountSum += 0.10;
  }
  
  const discountedMonthly = Math.round(monthlyWithSupport * (1 - discountSum));
  const yearlyTotal = discountedMonthly * 12 + oneTimeTotal;
  
  // 6. 사람이 읽기 좋은 breakdown
  const breakdown = [
    `추천 플랜: ${plan}`,
    `기본료: ${rate.base.toLocaleString()}원/월`,
    empOver > 0 ? `직원 초과 ${empOver}명 × ${rate.empOver.toLocaleString()}원 = ${(empOver * rate.empOver).toLocaleString()}원/월` : '',
    vehOver > 0 ? `차량 초과 ${vehOver}대 × ${rate.vehOver.toLocaleString()}원 = ${(vehOver * rate.vehOver).toLocaleString()}원/월` : '',
    support247 > 0 ? `24/7 전담 지원: +${support247.toLocaleString()}원/월` : '',
    discounts.length > 0 ? `할인 합계: -${(discountSum * 100).toFixed(0)}%` : '',
    `월 결제액(할인 후): ${discountedMonthly.toLocaleString()}원`,
    oneTimeFees.length > 0 ? `1회성 비용: ${oneTimeTotal.toLocaleString()}원` : '',
    `연간 총액: ${yearlyTotal.toLocaleString()}원`,
  ].filter(Boolean);
  
  return {
    recommendedPlan: plan,
    monthlyBase: rate.base,
    monthlyExtra,
    monthlyTotal: monthlyWithSupport,
    oneTimeFees,
    oneTimeTotal,
    discounts,
    discountedMonthly,
    yearlyTotal,
    breakdown,
  };
}
```

## 7. 자동 이메일 발송

```typescript
// lib/mailer.ts
import nodemailer from 'nodemailer';

const transporter = nodemailer.createTransport({
  host: process.env.SMTP_HOST,
  port: 587,
  auth: { user: process.env.SMTP_USER, pass: process.env.SMTP_PASS },
});

export async function sendQuoteEmail(opts: {
  to: string;
  applicationNo: string;
  pdfUrl: string;
  recommendedPlan: string;
  monthlyAmount: number;
}) {
  await transporter.sendMail({
    from: '"CleanERP 영업팀" <sales@helpbiz.kr>',
    to: opts.to,
    subject: `[CleanERP] 견적서 발송 — ${opts.applicationNo}`,
    html: `
      <p>안녕하세요. CleanERP 견적서를 발송드립니다.</p>
      <ul>
        <li>접수번호: ${opts.applicationNo}</li>
        <li>추천 플랜: <strong>${opts.recommendedPlan}</strong></li>
        <li>월 결제액: <strong>${opts.monthlyAmount.toLocaleString()}원</strong></li>
      </ul>
      <p><a href="${opts.pdfUrl}">견적서 PDF 다운로드</a></p>
      <p>견적은 30일간 유효합니다.</p>
    `,
    attachments: [{ path: opts.pdfUrl }],
  });
}
```

`.env.prod`에 SMTP 설정 추가:
```
SMTP_HOST=smtp.gmail.com
SMTP_USER=sales@helpbiz.kr
SMTP_PASS=...
```
