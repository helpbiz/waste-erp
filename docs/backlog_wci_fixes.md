## 버그 수정

### #1. 민원 처리완료 후 사진 촬영/앨범 업로드 오류
- **증상**: 처리완료 상태에서 사진 촬영 또는 앨범 업로드 시 오류 발생
- **관련 파일 추정**: 민원 처리 완료 폼 / 사진 업로드 컴포넌트
- **우선순위**: 높음 (현장 업무 직결)

### #6. 근로자 민원등록 후 내 민원처리 현황 데이터 누락
- **증상**: 민원 등록 후 조회 시 사진·연락처 등 일부 정보 소실
- **요청**: 관리자와 동일하게 전체 데이터(본인 포함) 유지 및 표시
- **우선순위**: 높음

## 기능 개선

### #2. 결재관리 — 승인 후 수정·취소 기능 추가
- **현재**: 승인 완료 시 수정·취소 불가
- **요청**: 승인 후에도 수정 및 취소 가능하도록 권한 추가
- **참고**: 권한 범위 (CONTRACTOR_ADMIN / INTERNAL_ADMIN 등) 결재 시 확인 필요

### #3. 민원 처리완료 건 수정 시 상태 변경 가능
- **현재**: 처리완료 수정 시 민원유형만 변경 가능
- **요청**: 상태 선택지 추가
  - 수거완료 / 수거대기 / 처리중 / 보류
- **관련 파일 추정**: 민원 수정 폼 컴포넌트

### #4. 태그된 담당자 알림 개선 (공지사항 / 민원관리)
- **현재**: 주소 및 내용만 표시
- **요청**: 관리자 화면과 동일하게 전체 정보 표시
  - 사진, 전화번호, 담당자, 처리메모 등 전체 항목
- **관련**: Push 알림 페이로드 확장 또는 알림 클릭 시 상세 화면 연동

### #5. 근로자 민원등록 — 지도 선택 → 주소 직접 입력 추가
- **현재**: 지도에서 위치 선택 방식만 지원
- **요청**: 주소 직접 입력 방식 추가 (관리자 기능과 동일하게)
- **관련 파일 추정**: 근로자 민원 등록 폼

## 차량일지 개선

### #7-A. 근로자 — 차량일지 작성 후 내역 확인 기능
- **현재**: 작성 후 확인 화면 없음
- **요청**: 제출한 차량일지 내역 조회 가능하도록 표시

### #7-B. 관리자 — 차량일지 상세 정보 전체 노출
- **현재**: 차량·기사·주행거리·운행시간·점검항목만 표시
- **요청**: 작업내역 포함 전체 항목 표시
  - routeDetail JSON의 모든 항목 (연료·정비·특이사항 등)
