## 1. 프로젝트 개요

### 도메인
지자체(시·구청)와 위탁업체가 함께 사용하는 **생활폐기물 수집·운반 ERP**.
실제 환경미화원·청소차 운영을 디지털로 관리.

### 사용자 그룹 (5단계 RBAC)
| Role | 권한 |
|---|---|
| `SUPER_ADMIN` | 전체 시스템 관리, 모든 지자체·업체 데이터 |
| `MUNI_ADMIN` | 자기 지자체 산하 위탁업체만 |
| `CONTRACTOR_ADMIN` | 자기 위탁업체 직원만 |
| `INTERNAL_ADMIN` | 위탁업체 내부 관리자 (팀장 등) |
| `WORKER` | 본인 데이터만 |

### 모듈 (12개)
1. 사용자 관리 (직책 12종, 부서, 사진/서명 업로드)
2. 근태 관리 (출퇴근, 월별 집계)
3. 휴가 관리 (11종 휴가, 2단계 결재, 캘린더)
4. 결재 정책 (직책별 매트릭스, 위임)
5. 안전관리 (TBM 서명, 일별 출력)
6. 실적 관리 (생활폐기물·자원순환·대형폐기물)
7. 민원 (대형폐기물 자동 연동: '빼기' 앱)
8. 차량 관리 (운행일지, 점검)
9. 실시간 차량조회 (Leaflet+OSRM 도로 스냅 + TSP 최적화)
10. 통계·보고서 (서명 워터마크, 인쇄)
11. PWA (앱 설치 버튼)
12. 슈퍼관리자 콘솔 (지자체 권한 정책 + 차고지 관리)

## 2. 기술 스택

### Frontend
- **Next.js 14.2.5** (App Router, standalone build)
- **React 18.3.1** + **TypeScript**
- **TailwindCSS** + **shadcn/ui**
- **react-leaflet 4.2.1** (지도) + **leaflet.heat** (히트맵)

- **AES-256-GCM** PII 암호화 (`lib/crypto.ts`, KMS abstraction: local/aws/vault)

### 개발 도구
- **Claude Code** + **bkit v2.1.0** (AI Native PDCA)
- **gap-detector**, **pdca-iterator** Agent 활용
