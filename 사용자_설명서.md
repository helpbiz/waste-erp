## 1. 이게 뭐예요?

🚛 **쓰레기차 회사**에서 쓰는 컴퓨터 프로그램이에요.

우리 동네 쓰레기차가 어디를 다니고, 누가 일을 하고, 휴가는 며칠 남았는지...
**이런 걸 컴퓨터가 다 기억해 주는 마법 책**이에요.

```
   🏢 우리 회사
   ├─ 🚚 쓰레기차 (어디 다녀요?)
   ├─ 👷 일하는 사람 (오늘 누가 와요?)
   ├─ 🏖️ 휴가 (며칠 남았어요?)
   ├─ 📋 안전 약속 (다친 사람 있어요?)
   ├─ 📊 일한 양 (얼마나 치웠어요?)
   └─ 📞 민원 (시민이 부탁한 일)
```

## 2. 누가 쓰는 프로그램이에요?

여러 사람이 같이 써요. **사람마다 보이는 화면이 달라요!**

| 누구? | 어떤 사람? | 뭘 봐요? |
|---|---|---|
| 👑 **슈퍼관리자** | 회사 만든 사람 | 모든 걸 다 볼 수 있어요 |
| 🏛️ **지자체 관리자** | 시청·구청 사람 | 우리 동네 쓰레기차만 봐요 |
| 🏢 **회사 사장님** | 위탁업체 대표 | 우리 회사 직원만 봐요 |
| 👔 **관리자** | 팀장님·실장님 | 직원 출근·휴가 봐요 |
| 👷 **운전기사·수거원** | 차 모는 사람 | 자기 출근·휴가만 봐요 |

> 🎭 **비유**: 학교에서 **선생님**, **교장 선생님**, **학생**이 보는 게 다른 것처럼요!

## 3. 어떻게 들어가요? (로그인)

```
┌────────────────────────┐

│  🔒 비밀번호 [        ]│
│                        │
│      [ 들어가기 ]      │
└────────────────────────┘
```

## 4. 메뉴마다 뭘 해요?

화면 **왼쪽**에 메뉴가 줄줄이 있어요. 하나씩 알려줄게요.

### 🏠 메인 대시보드
> "오늘 회사가 어떻게 돌아가는지 한눈에 보는 곳"

- 오늘 출근한 사람 몇 명?
- 휴가 신청한 사람 있어요?
- 차 몇 대가 일하고 있어요?

```
   📊 오늘의 회사
   ┌─────┬─────┬─────┐
   │ 👥  │ 🚚  │ 🏖️  │
   │ 32명 │ 8대  │ 3건  │
   └─────┴─────┴─────┘
   출근   움직중   휴가
```

### 👥 사용자 관리
> "직원 명단 책"

- 직원 새로 들어왔어요? → **등록** 버튼
- 직책 바꿨어요? → **수정** 버튼
- 사진·서명도 같이 저장돼요 ✏️

### ⏰ 근태 관리
> "출근 도장 찍는 곳"

```
   🌞 출근 ─────────► 🌙 퇴근
   "오늘 9시 1분에 왔어요!"
```

### 🏖️ 휴가 관리
> "쉬는 날 신청하는 곳"

휴가 종류가 **11가지** 있어요:
- 연차 (1년에 며칠 쉴 수 있어요)
- 반차 (반나절만 쉬어요)
- 병가 (아플 때)
- 출산휴가, 가족돌봄, 생리휴가...

```
   📅 휴가 신청
   ┌──────────────┐
   │ 언제부터?    │
   │ 언제까지?    │
   │ 왜요?        │
   │              │
   │ [신청하기]   │
   └──────────────┘
   ↓
   👔 팀장님 승인 ✅
   ↓
   👑 사장님 승인 ✅
   ↓
   🎉 휴가 확정!
```

### 🚛 실시간 차량조회 ⭐
> "우리 회사 쓰레기차가 지금 어디 있는지 보는 지도"

- 지도에 차들이 점으로 떠 있어요 🔵
- 차고지(차 두는 곳)에서부터 어디로 가야 좋은지 **컴퓨터가 길을 그려줘요!**
- 빨간 점 = **출발지** 🔴
- 보라 점 = **도착지** 🟣
- 하늘색 점 = **중간 정류장** 🔷

```
   🗺️  지도 화면
   ┌───────────────────┐
   │  🏁(차고지)       │
   │   ↘             │
   │    1️⃣            │
   │     ↘           │
   │      2️⃣ ─► 3️⃣    │
   │            ↘    │
   │             4️⃣🏁│
   └───────────────────┘
   "이 순서대로 가면 가장 빨라요!"
```

### 📋 안전관리
> "다치지 말자고 약속하는 곳"

매일 아침 **TBM**이라는 회의를 해요. 거기서 **서명**(이름 사인)을 받아요.
서명 안 한 사람 누구인지도 보여줘요 ⚠️

### 📊 실적 관리
> "오늘 쓰레기 얼마나 치웠는지 적는 곳"

- 생활폐기물 (집에서 나오는 쓰레기)
- 대형폐기물 (소파·침대 같은 큰 거)
- 자원순환센터에 보낸 양

### 📞 민원
> "시민이 부탁한 일 받는 곳"

> "우리 집 앞에 큰 쓰레기 있어요!"
> 라고 시민이 부탁하면 여기에 들어와요.

대형폐기물은 **'빼기' 앱**에서 자동으로 들어와요 📱

### 📈 통계·보고서
> "한 달에 얼마나 일했는지 그림으로 보는 곳"

도장 찍어서 **인쇄**할 수도 있어요 🖨️

## 5. 자주 쓰는 5가지

### ① 출근하기

```
   🏠 회사 도착 → 핸드폰 꺼내기
   → 🌐 wci.helpbiz.kr 들어가기
   → 👤 로그인
   → ⏰ 근태 메뉴
   → ✅ 출근 버튼 누르기 → 끝!
```

### ② 휴가 신청하기

```
   🏖️ 휴가 메뉴
   → 신청 버튼
   → 날짜·이유 쓰기
   → 보내기
   → 👔 팀장님이 OK 누르면 끝!
```

### ③ 내 정보 보기

```
   👤 내 프로필 메뉴
   → 사진·전화번호 바꾸기
   → 비밀번호 바꾸기 (자주 바꾸세요!)
```

### ④ 차고지 정해주기 (사장님 전용)

```
   ⚙️ 슈퍼관리자 → 회사정보·차고지 탭
   → 주소 쓰기 ("서울시 강남구...")
   → 🔍 [검색] 누르기 → 자동으로 좌표 찾아줘요
   → 저장
   ✨ 이제 모든 차가 여기서 출발해요!
```

### ⑤ 가장 빠른 길 찾기

```
   🚛 실시간 차량조회 → 추천경로 메뉴
   → 가야 할 곳들 점 찍기 (1, 2, 3, 4...)
   → [경로 계산] 버튼
   → 컴퓨터가 가장 빠른 순서로 길을 그려줘요 🗺️
```

## 6. 핸드폰에 앱처럼 깔기

> 우리 프로그램은 핸드폰에 **앱처럼 설치**할 수 있어요!

### 안드로이드 폰

1. 크롬으로 `https://wci.helpbiz.kr` 열기
2. 화면 아래에 **"앱 설치"** 버튼이 떠요
3. 누르면 → 끝! 핸드폰 화면에 아이콘이 생겨요 📱

### 아이폰

1. 사파리로 `https://wci.helpbiz.kr` 열기
2. 아래쪽 **공유 버튼** (네모에 위 화살표 ⬆️) 누르기
3. **"홈 화면에 추가"** 누르기 → 끝!

## 7. 도와주세요!

### 💬 자주 있는 일

| 이런 일이? | 이렇게 해요 |
|---|---|
| 비밀번호 까먹었어요 😢 | 사장님께 말씀드리세요 |

회사 사장님이나 관리자분께 직접 말씀드려 주세요.
